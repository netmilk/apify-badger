# apify-badger
Generate README badges for public Apify Actors

# Usage

![screenshot](https://github.com/netmilk/apify-badger/blob/main/screenshot.png?raw=true)

## Install

```
$ git clone https://github.com/netmilk/apify-badger.git
$ cd apify-badger
$ npm install
$ node index.js
$ Started on port 8080
```
